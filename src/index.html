RS Lang поможет изучить английский

RS lang– приложение для изучения иностранных слов, включающее электронный учебник с базой слов для изучения, мини-игры для их повторения, страницу статистики для отслеживания индивидуального прогресса.

О приложении

См. видео

Оцените всепреимущества приложения

Учебник

Более 3500 слов для изучения, рабитых на разделы по уровню твоей подготовки с удобной навигацией

Игры

2 увлекательные игры на развитие запоминания слов и восприятия на слух

Статистика

Отслеживайте свой прогресс в индивидуальной статистике, ставь цели и вдохновляйся на достижения новых результатов каждый день
